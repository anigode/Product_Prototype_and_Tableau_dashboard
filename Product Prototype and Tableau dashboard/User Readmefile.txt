Project Title: Dashboard for the Football Players (Messi, Ronaldo) Performance Comparison

Description:  This project uses the R code and Tableau and produces the Dashboard for the entertainment domain

Dependencies & Installation:  Install Windows 8,10, Tableau desktop software, R studio, R programming, Install R packages

Dataset & Code: It is given along with the project

Execution of the Program:

1) run the R code to import the data with R studio
2) Clean the data using R
3) Make some visualisation
4) Import the Tableau ".twbx" file into Tableau and see the Dashboard
